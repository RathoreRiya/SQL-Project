# PostgreSQL Music Store Data Analysis

## Project Overview
This project is based on a Music Store database analysis using PostgreSQL.  
The goal of this project is to analyze customer behavior, sales trends, music genres, artists, and invoice data using SQL queries.

The project includes beginner to intermediate SQL concepts such as:
- Joins
- Aggregate Functions
- GROUP BY
- ORDER BY
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Recursive Queries

## Dataset Used
Music Store / Chinook Database

---

## Project Objectives

This project answers business-related questions such as:

1. Who is the senior most employee?
2. Which countries generate the most invoices?
3. What are the top invoice totals?
4. Which city generates the highest revenue?
5. Who is the best customer?
6. Find all Rock music listeners.
7. Top 10 Rock bands by track count.
8. Songs longer than average song length.
9. Amount spent by each customer on artists.
10. Most popular music genre for each country.
11. Top customer by country based on spending.
